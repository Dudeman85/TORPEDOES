46/*
47 Opens zip archive with compression level using the given mode.
48
49 Args:
50 zipname: zip archive file name.
51 level: compression level (0-9 are the standard zlib-style levels).
52 mode: file access mode.
53 'r': opens a file for reading/extracting (the file must exists).
54 'w': creates an empty file for writing.
55 'a': appends to an existing archive.
56
57 Returns:
58 The zip archive handler or NULL on error
59*/
60extern struct zip_t *zip_open(const char *zipname, int level, char mode);

70/*
71 Opens an entry by name in the zip archive.
