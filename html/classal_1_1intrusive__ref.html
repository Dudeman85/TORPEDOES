bool al::intrusive_ref< T >::releaseIfNoDelete ( )
inlinenoexcept

Release only if doing so would not bring the object to 0 references and delete it. Returns false if the object could not be released.

NOTE: The caller is responsible for handling a failed release, as it means the object has no other references and needs to be be deleted somehow.
