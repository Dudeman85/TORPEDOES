Sets the filter state for the specified filter type and its parameters.

Parameters
typeThe type of filter to apply.
f0normThe normalized reference frequency (ref / sample_rate). This is the center point for the Shelf, Peaking, and BandPass filter types, or the cutoff frequency for the LowPass and HighPass filter types.
gainThe gain for the reference frequency response. Only used by the Shelf and Peaking filter types.
bandwidthNormalized bandwidth of the transition band.
